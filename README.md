# Number Recognition

A Deep Learning project that identifies handwritten digits using a Convolutional Neural Network (CNN). 

## Overview
This repository contains a Jupyter Notebook (`Number_recognition.ipynb`) that builds, trains, and evaluates a digit classification model using the **MNIST dataset**. 

## Tech Stack
* Python
* TensorFlow / Keras
* NumPy
* Matplotlib & Seaborn

## Model Performance
The CNN model is trained over 10 epochs and achieves an impressive accuracy of over **99%** on the validation/test data.
